
AST klasa:
SelectStatement

Gramatičko pravilo:
SelectStatement → select begin CaseList end
CaseList → Case
Case → case Expression => Statement


Primer izvornog koda (Primer.mp):

program
begin
    a : integer;
    a := 1;

    select
    begin
        case a > 0 =>
            a := 2;
    end
end.


Opis generisanog međukoda:

1. Inicijalizacija promenljive

Najpre se izvršava naredba dodele vrednosti promenljivoj a:

a := 1;

Generisani međukod:

Load_Const  R1, 1
Store       R1, a


2. Obrada SelectStatement naredbe

SelectStatement sadrži jedan Case deo. Za svaki Case deo posebno
se izračunava uslov, a zatim se uslovno izvršava njegovo telo.


3. Izračunavanje uslova Case dela (a > 0)

Izraz a > 0 se prevodi u sledeći međukod:

Load_Mem    R1, a
Load_Const  R2, 0
Cmp_GT      R1, R2

Instrukcija Cmp_GT poredi vrednosti u registrima R1 i R2.
Ako je uslov ispunjen (a > 0), u registar R1 se upisuje vrednost 1,
u suprotnom se upisuje 0.

Rezultat uslova se čuva u privremenoj promenljivoj:

Store       R1, $tmp0


4. Provera uslova i uslovni skok

Na osnovu vrednosti izračunatog uslova vrši se uslovni skok:

Load_Mem    R1, $tmp0
JumpIfZero  R1, lab0

Ako je vrednost uslova jednaka nuli (uslov nije ispunjen),
izvršavanje se nastavlja od labele lab0, čime se preskače telo Case dela.


5. Telo Case dela

Ako je uslov ispunjen, izvršava se naredba:

a := 2;

Generisani međukod:

Load_Const  R1, 2
Store       R1, a


6. Završetak SelectStatement naredbe

Labela lab0 označava kraj obrade Case dela i kraj SelectStatement-a:

lab0:


7. Završetak programa

Na kraju programa generiše se instrukcija:

HALT
